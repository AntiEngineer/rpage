Responsive pagination for Bootstrap

rpage is an easy to use plugin for making Bootstrap 3's pagination more responsive.

Usage

Just include responsive-pagination.js and call the rPage function on the pagination element like this:

$(document).ready(function () {
    $(".pagination").rPage();
});
